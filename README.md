# Scoop bucket for ZeroDDS

Install the ZeroDDS toolchain on Windows via Scoop:

```powershell
scoop bucket add zerodds https://github.com/zero-objects/scoop-zerodds
scoop install zerodds
```

This bucket is auto-updated by the [zerodds release pipeline](https://github.com/zero-objects/zero-dds/blob/main/.github/workflows/release.yml).
